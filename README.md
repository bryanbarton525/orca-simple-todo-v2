# Todo App

A minimal single-page todo application built with Next.js 15+, TypeScript, Tailwind CSS and the new App Router. It demonstrates adding, completing and deleting tasks with persistence in localStorage. The project is ready to run locally and serves as a starter template for small frontend projects.

## Quickstart

```bash
# Install dependencies
pnpm install

# Run the dev server
pnpm dev
```

Open your browser at http://localhost:3000 to see the app. The source is in `app/` and uses only vanilla JS/TS, CSS and Tailwind utility classes.

You can customize the styles, add persistence backends or deploy to Vercel with zero configuration.

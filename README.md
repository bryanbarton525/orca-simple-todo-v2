# Minimal Todo App

A tiny single‑page todo application built with vanilla HTML, CSS, and JavaScript.

## Features
- Add todos with a category (Work or Personal)
- Mark todos as complete
- Delete todos
- Filter the list by category using checkboxes
- Persistence via `localStorage`

## Running the app locally
1. Ensure you have a recent browser installed.
2. Open the file `index.html` in the browser.
   - You can simply double‑click the file or run a static server, e.g.:
   ```bash
   npx http-server .
   ```
3. The app should load; you can start adding todos right away.

## Build
The project includes a minimal `build` script to satisfy the npm build requirement. Running `npm run build` will output a short message and exit successfully.

```bash
npm run build
```

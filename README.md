# Todo App\n\nThis is a minimal single‑page Todo application built with Next.js 14, TypeScript, and Tailwind CSS.\n\n## Development\n\n```bash\nnpm install\nnpm run dev\n```\n\nOpen http://localhost:3000 to view the app.\n\n## Environment Variables\n\nAdd a `.env.local` file at the project root:\n\n```dotenv\n# Database\nDB_URI=postgres://user:pass@localhost:5432/db\n\n# SMTP\nSMTP_HOST=smtp.example.com\nSMTP_PORT=587\nSMTP_USER=you@example.com\nSMTP_PASS=secret\n\n# Discord\nDISCORD_TOKEN=YOUR_DISCORD_BOT_TOKEN\n\n# VAPID Keys\nVAPID_PUBLIC_KEY=YOUR_PUBLIC_KEY\nVAPID_PRIVATE_KEY=YOUR_PRIVATE_KEY\n```\n\n## Scripts\n\n- `npm run dev` – start development server\n- `npm run build` – build for production\n- `npm run start` – start the production server\n- `npm run test` – run tests (currently none)\n- `npm run seed` – run Prisma seed script

# Todo App

## Overview

A minimal single‑page todo application built with plain HTML, CSS, and JavaScript. Todos are persisted in `localStorage`.

## Features

- Add a todo
- Mark a todo as complete
- Delete a todo
- Persist across page reloads

## File Structure

```
index.html
style.css
app.js
README.md
```

## Running the App

1. Clone the repository.
2. Open `index.html` in a browser. No build or server is required.

```sh
# If you prefer a local web server:
python -m http.server
```

## Development

No Node tooling is required. If you want to run a simple server:

```sh
# Using Python
python -m http.server
```
